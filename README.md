# AutoParts

Интернет-магазин автозапчастей на фреймворке Yii 2.

## Возможности

- Каталог автомобилей с выбором марки и модели
- Фильтрация по типу двигателя (бензин, дизель, гибрид) и приводу (полный, передний)
- Асинхронная фильтрация через AJAX
- Компонентная структура представлений

## Технологии

- **Backend**: Yii 2, PHP 7+
- **Frontend**: jQuery, Webpack, SCSS, Babel
- **База данных**: MySQL

## Установка

```bash
composer install
npm install
```

## Настройка

1. Скопируйте конфигурацию БД:
```bash
cp config/db.php config/db.local.php
```

2. Отредактируйте `config/db.local.php` с настройками подключения к БД.

3. Примените миграции:
```bash
php yii migrate
```

## Запуск

### Веб-сервер

```bash
php yii serve
```

### Frontend сборка

```bash
# Development
npm run dev

# Production
npm run build
```

## Структура проекта

```
app/
├── config/          # Конфигурация приложения
├── controllers/     # Контроллеры
├── models/          # Модели данных
├── views/           # Представления
│   ├── components/  # Переиспользуемые компоненты
│   ├── layouts/    # Шаблоны страниц
│   └── site/       # Страницы сайта
web/
├── css/             # Скомпилированные стили
├── js/              # Скомпилированные скрипты
├── scss/            # Исходники стилей
└── dist/            # Бандлы webpack
```

## Маршруты

- `/` — Главная страница
- `/catalog` — Каталог (полный)
- `/catalog/<brand>` — Каталог марки
- `/catalog/<brand>/<model>` — Каталог модели
